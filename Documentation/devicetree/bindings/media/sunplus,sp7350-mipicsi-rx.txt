Sunplus SP7350 MIPI/CSI RX controller
-------------------------------------

The device node of Sunplus SP7350 MIPI/CSI RX controller has following
properties.

Required properties:
 - compatible: should be "sunplus,sp7350-mipicsi-rx".
 - reg: addresses and length of the register sets for MIPI CSI
        controller.
 - reg-names: one name for each entry in the reg property.
 - clocks: clock selector for MIPI CSI controller.
 - resets: reset controller selector for MIPI CSI controller.
 - pinctrl-names: Names corresponding to the numbered pinctrl states.
 - pinctrl-0: phandle for the 'default' state of pin configuration.

Example:

mipicsi0: mipicsirx@0xf8005280 {
	compatible = "sunplus,sp7350-mipicsi-rx";
	reg = <0x0 0xf8005280 0x0 0x80>;
	reg-names = "mipicsi";
	clocks = <&clkc MIPICSI0>;
	clock-names = "clk_mipicsi";
	resets = <&rstc RST_MIPICSI0>;
	reset-names = "rstc_mipicsi";
	sunplus,id = <0>;
};
